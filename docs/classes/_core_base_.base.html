  • chunkRetry: number

    最大重试次数

  • chunkSize: number
  • concurrency: number

    并发上传数（块级上传，非必须,默认如下）

  • taskConcurrencyInWorkers: number

    可并发执行任务数（非必须，默认3即可） 此参数主要在worker任务中起作用 基本表示一个worker线程在非忙时可承担的最大任务数 设置大了意义不大
